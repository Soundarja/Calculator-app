# Python GUI Calculator

A simple GUI-based arithmatic calculator application built using Python and Tkinter.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Error handling for division by zero
- Simple and modern GUI

## Technologies Used
- Python
- Tkinter

## How to Run
```bash
python python_calculator.py
```
